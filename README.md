# MERN Stack Starter Kit

This is a full-stack application using React, Redux Toolkit, Node.js, Express, MongoDB, and Redis.

### Setup

1. **Frontend**
   - Navigate to `frontend/`
   - Run `npm install`
   - Run `npm start`

2. **Backend**
   - Navigate to `backend/`
   - Run `npm install`
   - Run `node index.js`
